# RSAUtils

rsabuilder
==========

Script to create public and private key via openssl using "p", "q" and "e" in decimal value


Example:

Generate RSA keys with p=61, q=53 and e=17

```
$ cd 01-rsabuilder/
$ python rsabuilder.py 61 53 17
```
